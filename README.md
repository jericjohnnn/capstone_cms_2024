## ðŸš€ How to Install

### Backend
1. `cd backend`
2. `composer install`
3. `cp .env.example .env`
4. `php artisan key:generate`
5. `php artisan storage:link`
6. `php artisan migrate --seed` (or `php artisan migrate:fresh --seed`)
7. `php artisan serve`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### CMS
1. `cd cms`
2. `npm install`
3. `npm run dev`
